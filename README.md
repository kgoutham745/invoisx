# InvoisX - Invoice Management Website
Welcome to the InvoisX repository! This repository contains the code for an invoice management website called InvoisX. It is built using a Bootstrap template for the user interface, Datatables for data management, Firebase for authentication, real-time database, and storage capabilities. InvoisX provides a convenient solution for managing and organizing invoices for businesses.

## Features
**User Authentication**: InvoisX uses Firebase authentication to allow users to sign up, log in, and securely access their invoice data.

**Invoice Creation**: Users can create new invoices by filling in relevant details such as customer information, items, quantities, prices, and tax rates.

**Invoice Management**: InvoisX provides features to view, edit, and delete existing invoices.

**Invoice Search and Filtering**: Users can search for specific invoices using keywords or filter invoices based on criteria such as date, customer name, or invoice status.

**Real-time Data Updates**: Any changes made to invoices are instantly reflected in the database, ensuring real-time synchronization across multiple users and devices.

**File Storage**: InvoisX leverages Firebase storage to enable users to attach and manage supporting documents or files related to their invoices.

**Responsive Design**: The website is built using Bootstrap, ensuring a responsive and user-friendly experience across different screen sizes and devices.

## Usage:
To use InvoisX, follow these steps

-Clone the repository to your local machine using the following command: 
```
git clone https://github.com/kgoutham745/invoisx.git
```

-Set up Firebase authentication, real-time database, and storage by following the Firebase documentation.

-Configure your Firebase credentials in the appropriate files or environment variables.

-Open the index.html file in your preferred web browser.

-Sign up or log in to access the invoice management functionality.

-Create, view, edit, and delete invoices as needed.

-Utilize search and filtering options to find specific invoices.

-Attach supporting documents or files to invoices using the file storage feature.


## Dependencies:
InvoisX relies on the following dependencies:

**Bootstrap**: The website is built using Bootstrap for responsive design and styling.

**Datatables**: Datatables library is used to manage and display invoice data in a tabular format.

**Firebase**: Firebase is used for authentication, real-time database, and storage functionalities.
Ensure you have the necessary dependencies properly configured in your project.
